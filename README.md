## 搬瓦工是什么？一句话先说清楚

搬瓦工是一家 **VPS 主机服务商**，隶属于加拿大 IT7 Networks Inc.，成立于 2004 年，总部在加拿大。VPS 是"虚拟专用服务器"——你租一台远程服务器，有独立 IP、独立内存、独立磁盘，可以装系统、跑程序、建网站、做各种服务器端的事。

它的中文名"搬瓦工"是音译，BandWagonHost 的发音和"搬瓦工（bān wǎ gōng）"挺像的，所以圈子里叫顺了。

搬瓦工面向国内用户最大的卖点是：**专门针对中国大陆访问优化了路由线路**，特别是 CN2 GIA 这条线路，延迟低、晚高峰抗拥堵能力强，国内用户访问境外服务器时体验明显比普通线路好。

**Q：搬瓦工支持支付宝付款吗？**

支持。支付宝、PayPal、信用卡都可以，注册账号时信息建议真实填写，填假的地址可能会被系统判定为欺诈订单而自动退款。

**Q：搬瓦工是按月扣款还是年付一次？**

取决于你选择的计费周期。KVM 套餐有年付选项，CN2 GIA-E 有季付（$49.99）、半年付（$89.99）、年付（$169.99）可选，香港套餐最低月付。选越长的周期通常单价越低。

**Q：CN2 和 CN2 GIA 有什么区别？**

CN2 GT（普通 CN2）和 CN2 GIA 都是中国电信国际网络骨干，区别在于 GIA 是更高等级的线路，有专属带宽保障，晚高峰不和普通流量混用。实际表现上，GIA 晚高峰稳定性明显优于 GT，价格也更高。

**Q：搬瓦工 VPS 如果 IP 被封怎么办？**

搬瓦工提供付费换 IP 服务，不是免费的。换 IP 之后如果套餐支持多机房，可以同时把机房也切换一下，相当于双重刷新。CN2 GIA-E 套餐因为有多机房选项，这方面灵活度比香港套餐高。
